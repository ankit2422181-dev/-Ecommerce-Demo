# Ushm Ecommerce Demo

A basic ecommerce frontend prototype built using **HTML, CSS, and JavaScript** for my brand **Ushm Essence**, focused on luxury fragrances and gifting products.

## Features
- Product listing interface
- Add to Cart functionality
- Dynamic cart item counter
- Real-time total price calculation
- Responsive and clean UI design

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Project Purpose
This project was built to demonstrate frontend development skills and practical implementation of JavaScript for ecommerce product interactions. It simulates an online shopping experience for fragrance and gifting products.
